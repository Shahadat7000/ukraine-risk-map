# Ukraine Risk Map

Interactive geospatial risk visualization of Ukrainian cities based on:

- Combat proximity
- Strategic importance
- News intelligence
- Occupied territories

Built with **Python + Folium**.

---

## Features

- Risk prediction model
- Interactive map
- Frontline distance calculation
- Intelligence summaries
- Occupied territory detection

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Shahadat7000/ukraine-risk-map.git
cd ukraine-risk-map
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python final.py
```

The script generates:

```
ukraine_risk_map.html
```

Open the file in your browser to view the interactive map.
